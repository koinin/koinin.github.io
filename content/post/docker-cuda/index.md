# Docker-cuda

起因是想解决centos编译库太老旧的问题，把一些编译环境放在docker里面

编译这个flash_attn要编译好久好久



```sh
sudo docker run -it -d \
  --gpus all \
  --name my_cuda \
  --network host \
  -v /home/sunyatao_B:/root/sunyatao_B \
  --shm-size=1g \
  nvidia/cuda:11.8.0-runtime-ubuntu22.04 bash
  
sudo docker exec -it my_cuda bash

```
